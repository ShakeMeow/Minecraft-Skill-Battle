# Minecraft-Skill-Battle

## 專案介紹

這是我自行設計與製作的 Minecraft 技能對戰系統。

## 功能
17種不同技能
技能刷新與獲取系統
玩家淘汰機制
傷害與狀態效果系統
多張對戰地圖
隨機出生機制
技能平衡調整

## 使用技術

* Command Block
* Scoreboard
* Execute Command

## 作者

ShakeMeow


# 遊戲規則

玩家開始時會隨機出生於場地內。
每位玩家擁有 20 點生命值，死亡後即被淘汰，最後存活者獲勝。
玩家無法透過普通近戰造成傷害，所有傷害皆來自技能效果。
技能可透過場地中的技能箱獲得，每個技能使用後會消失，技能刷新點會定期重新產生新的技能。
